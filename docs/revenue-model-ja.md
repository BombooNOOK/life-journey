# 収益管理シート（シンプル版）

月10万円の利益目標に向けた、最小の管理テンプレートです。

## 1) 毎月入力する項目

- `A`: 新規鑑定注文数（件）
- `B`: 1件あたり平均売上（円）
- `C`: PDF再発行（有料300円）の購入件数（件）
- `D`: 固定コスト（円）
  - Vercel
  - Neon
  - Firebase
  - その他ツール費
- `E`: 変動コスト（円/件）
  - 決済手数料など

## 2) 計算式

- `売上` = `A * B + C * 300`
- `変動費` = `A * E`
- `営業利益` = `売上 - D - 変動費`
- `利益率` = `営業利益 / 売上`

## 3) 目標10万円の逆算

目標: `営業利益 >= 100,000`

必要売上の目安:

- `必要売上 >= 100,000 + D + (A * E)`

まず固定費を把握し、変動費を足してから「+10万円」で最低ラインを出します。

## 4) 参考シミュレーション

- `A=120件`
- `B=2,000円`
- `C=30件`
- `D=20,000円`
- `E=120円`

計算:

- 売上 = `120*2000 + 30*300 = 249,000`
- 変動費 = `120*120 = 14,400`
- 営業利益 = `249,000 - 20,000 - 14,400 = 214,600`

## 5) 運用ルール（毎週/毎月）

### 毎週

- Vercel / Neon / Firebase の使用量を確認
- PDF生成エラー率を確認

### 毎月

- このシートの `A〜E` を更新
- 次月の固定費見込み（`D`）を更新

## 6) アラートライン（推奨）

- 利益予測が `80,000円` を下回ったら、次を検討:
  - PDF無料回数を見直す（例: 3回 → 2回）
  - 再発行導線を強化（BASEリンクの視認性）
  - 重い処理（PDF）の実行回数を抑える導線を追加

## 7) 今回仕様との対応

- 鑑定書PDF無料回数: `3回`
- 4回目以降: 再発行（有料）導線へ

この制限は、原価の高いPDF生成コストを抑えるための主要なコントロールです。
